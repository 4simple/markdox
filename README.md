Markdox
========

Markdox is a documentation generator based on [Dox](https://github.com/visionmedia/dox) and Markdown.

Installation
---

    $ npm install -g markdox

Usage
---

    $ markdox myfile.js
    # or
    $ markdox myfile1.js myfile2.js

Quick Start
---
  
    $ markdox test/fixtures/a.js test/fixtures/b.js
